# Support

Use [GitHub Issues](https://github.com/Praxity/prax-proof/issues) for reproducible
bugs and documentation gaps. Include the Proof commit, deployment method,
relevant sanitized logs, and minimal synthetic reproduction data.

Proof is a free pre-release project without an SLA. Deployment-specific legal,
privacy, Cloudflare account, and infrastructure advice remains the operator's
responsibility. Use [private security reporting](SECURITY.md) for vulnerabilities.
