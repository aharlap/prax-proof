# Code of conduct

Participation in Proof's project spaces requires respectful, professional, and
constructive communication. Harassment, discrimination, threats, doxxing, and
publishing private learner or reporter information are not acceptable.

Focus criticism on technical decisions and observable behavior. Maintainers may
edit or remove content, close interactions, or restrict participation when
needed to protect contributors and users.

Report conduct concerns privately through the repository owner's GitHub contact
channels. Do not include confidential learner data in a conduct report.
